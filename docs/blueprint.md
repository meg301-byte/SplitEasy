# **App Name**: SplitEasy

## Core Features:

- Create Split Event: Allow users to create a new split event with a name.
- Add Participants: Enable users to add participants to the split event.
- Record Expenses: Allow users to record expenses with description, amount, and payer.
- Automatic expense sharing calculation: Calculate how much each person owes or is owed, with clear and easy-to-understand views for quick reconciliation.
- Settlement Tool: Suggest optimal transactions to minimize the number of payments required to settle debts.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and clarity.
- Background color: Very light gray (#F5F5F5), nearly white.
- Accent color: Pale violet-red (#D87093) for important actions and highlights, giving a touch of playfulness.
- Body and headline font: 'PT Sans' for a modern and readable look.
- Simple, clean icons for each action, like adding a participant or expense.
- Clear sections for debts and settlement to focus on core functionality.
- Subtle transitions and animations to give feedback for interactions.
